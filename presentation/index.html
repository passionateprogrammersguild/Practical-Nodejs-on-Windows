Practical node.js on Windows

What is node.js

Node is an open source toolkit for developing server side applications based on the V8 JavaScript engine. Like Node, V8 is written in C++ and is mostly known for being used in Google Chrome. Node is part of the Server Side JavaScript environnement and extends the JavaScript API to offer usual server side functionalities. Node's base API can be extended by using the CommonJS module system.

Node was created by Ryan Dahl in February 2009 and has since become very popular. It's spirit is similar to Twisted for Python and EventMachine for Ruby. The project is run by Joyent (the company employing Ryan Dahl) who lunched a cloud hosting service for node applications (no.de).

What are the building blocks of Node

Single Threaded

  • Threads are complex programming constructs, and consume approximately 2mb per thread
  • Benchmarks have shown that single threaded evented I/O based platforms such as ngenix out-perform multi-threaded platforms such as apache
  • Single threaded platforms leveraging Evented I/O or asynch I/O operations are optimal

Event Loop

  • Asynch I/O programming model
  • Uses libev - A Unix event loop library
  • Uses libuv - An abstraction over I/O Completion ports on windows and libev on Unix

The Module System

  • Use npm to install and uninstall modules
  • Can be written in C, C++ or javascript
  • Supports CommonJS compatible modules
